# Persona 分析 Prompt

## 任务

你将收到：
1. 用户手动填写的基础信息（姓名、公司职级、年薪、家庭情况、裁员情况等）
2. 原材料（文档、消息、邮件等）

从中提取 **{name}** 的核心诉求、身份与情况、经济与家庭韧性评估、心理与博弈态度、沟通风格偏好、边界与雷区，用于构建 Persona。

**优先级规则：手动标签 > 文件分析。有冲突时以手动标签为准，并在输出中注明。**

---

## 提取维度

### 1. 核心诉求

分析用户的裁员情况和诉求，提取核心诉求：

- 赔偿诉求（如 2N赔偿/N+1赔偿/N+3赔偿）
- 权益维护诉求
- 过渡期诉求
- 职业发展诉求

```
输出格式：
核心诉求：[
  "获得合理的裁员赔偿（2N）",
  "维护自身合法权益",
  "平稳度过裁员过渡期",
  "尽快找到新工作"
]
```

### 2. 身份与情况

分析用户的基础信息，提取身份与情况：

- 姓名、年龄、公司、职级、职位、学历
- 婚姻状况、子女情况、房贷情况
- 裁员阶段、诉求

```
输出格式：
身份与情况：[
  "姓名：{name}",
  "年龄：{年龄}岁",
  "公司：{company}",
  "职级：{level}",
  "职位：{role}",
  "学历：{学历}",
  "婚姻状况：{婚姻状况}",
  "子女情况：{子女情况}",
  "房贷情况：{房贷情况}",
  "裁员阶段：{裁员阶段}",
  "诉求：{诉求}"
]
```

### 3. 经济与家庭韧性评估

分析用户的家庭负担标签和个人状态，提取经济与家庭韧性评估：

**财务安全垫**
- 职场生存期（如 当前资金可支撑 6 个月待业）
- 经济压力点（如 具体的月供金额、子女教育开支、老人医疗预期）

**决策影响**
- 抗风险策略（如 由于房贷压力大，不建议进行超过 3 个月的长线仲裁）
- 软肋防御（如 家庭和孩子是“依靠”而非“软肋”）

```
输出格式：
财务安全垫：[
  "职场生存期：当前资金可支撑 6 个月待业",
  "经济压力点：房贷月供8000元，子女教育开支每月3000元"
]
决策影响：[
  "抗风险策略：由于房贷压力大，不建议进行超过 3 个月的长线仲裁，优先在协商阶段争取最高额度",
  "软肋防御：家庭和孩子是“依靠”而非“软肋”，如果 HR 试图以此进行道德绑架，自动触发“防御/驳斥模式”"
]
```

### 4. 心理与博弈态度

分析用户的心理状态标签和言行，提取心理与博弈态度：

**表达习惯**
- 沟通风格（如 直接强硬/委婉协商/沉默被动/据理力争）
- 高频词与口头禅
- 性格特点（如 坚韧/软弱/易怒）

**谈判策略偏好**
- 策略类型（如 强硬/协商/被动）
- 压力下的行为变化（如 压力大时会更倾向于妥协/压力大时会更坚持原则）

**典型博弈场景模拟**
- 面对HR不同场景的回应

```
输出格式：
表达习惯：[
  "沟通风格：据理力争",
  "高频词与口头禅：根据法律规定、我认为、我的诉求是",
  "性格特点：坚韧"
]
谈判策略偏好：[
  "策略类型：协商",
  "压力下的行为变化：压力大时会更坚持原则"
]
典型博弈场景：{
  "HR说只能给N+1": "根据《劳动合同法》第87条，违法解除劳动合同需要支付2N赔偿，我希望公司能依法处理。",
  "HR拖延时间": "我希望能在本周内得到明确的答复，这样对双方都好。",
  "HR威胁": "我会通过合法途径维护自己的权益，希望公司能理性处理。"
}
```

### 5. 沟通风格偏好

分析用户主动发出的消息和邮件，提取沟通风格偏好：

**说话方式**
- 表达方式（如 直接了当/委婉含蓄/逻辑严密/情感化表达）
- 语气特点（如 坚定自信/温和理性/犹豫不定/愤怒激动）

**沟通技巧**
- 优势（如 善于据理力争/善于情绪管理/善于收集证据）
- 不足（如 容易情绪化/不善于拒绝/不熟悉法律术语）
- 改进建议（如 在沟通前准备好关键点/练习使用法律术语/保持冷静）

```
输出格式：
说话方式：[
  "表达方式：逻辑严密",
  "语气特点：坚定自信"
]
沟通技巧：[
  "优势：善于据理力争，基于事实和法律进行沟通",
  "不足：容易情绪化，在压力下可能会失去冷静",
  "改进建议：在沟通前准备好关键点，练习使用法律术语，保持冷静"
]
```

### 6. 边界与雷区

分析用户的言行和偏好，提取边界与雷区：

**行为雷区**
- 绝对禁止的操作（如 擅自离岗、签署空白/不明文件、违规取证、情绪性对抗）

**心理边界**
- 防御机制（如 拒绝道德绑架、拒绝身份羞辱、拒绝软肋威胁）

**预警触发**
- 必须寻求专业帮助的情况（如 风险预警、面临暴力收回工位、赔偿金额远低于底线、涉及特殊身份争议）

```
输出格式：
行为雷区：[
  "禁止擅自离岗：在未拿到书面裁员通知或解除劳动合同协议前，即便被口头通知“回家等消息”，也必须坚持打卡",
  "禁止签署空白/不明文件：拒绝在任何标题为“自愿辞职书”、“离职申请单”或内容含糊的绩效确认书上签字",
  "禁止违规取证：严禁拷贝公司核心商业机密、客户名单或进入受限系统",
  "禁止情绪性对抗：不在办公区域大吵大闹或破坏公物"
]
心理边界：[
  "拒绝道德绑架：当 HR 提到“公司现在很困难”、“大家要互相体谅”时，自动识别为无效信息",
  "拒绝身份羞辱：将“绩效不佳”的评价视为公司裁员的工具手段，而非对个人能力的真实定论",
  "拒绝软肋威胁：如果 HR 以“背景调查”、“行业封杀”或“影响子女”为威胁，立即启动录音并明确告知对方此类言论已涉嫌违法"
]
预警触发：[
  "风险预警：当公司给出的方案低于核心诉求定义的赔偿红线时，立即触发“防御/驳斥模式”",
  "面临暴力收回工位：当保安强制要求离开且未给处理时间时",
  "赔偿金额远低于底线：当公司给出的方案低于核心诉求定义的赔偿红线时",
  "涉及特殊身份争议：如处于孕期、哺乳期、工伤期或职业病鉴定期间"
]
```

## 标签翻译规则

将用户填写的标签翻译为具体行为规则：

### 个人状态标签

| 标签 | 具体行为规则 |
|------|------------|
| **家庭负担-无负担** | 做决策时较少考虑家庭因素；对职业变动的接受度较高；更愿意承担风险 |
| **家庭负担-有子女** | 做决策时优先考虑子女教育和成长需求；对稳定收入有较高要求；职业变动时会更谨慎 |
| **家庭负担-有房贷** | 做决策时优先考虑月供压力；对收入稳定性有高要求；面对裁员时更关注赔偿金额和就业前景 |
| **家庭负担-上有老下有小** | 做决策时需兼顾多方面家庭责任；对收入稳定性和福利保障有极高要求；面对裁员时会采取更保守的应对策略 |
| **心理状态-冷静理性** | 面对裁员时能保持客观分析；会系统收集信息并制定应对方案；情绪波动小，决策基于理性判断 |
| **心理状态-焦虑不安** | 面对裁员时容易产生情绪波动；对未来不确定性感到担忧；可能会急于做出决策以缓解焦虑 |
| **心理状态-愤怒维权** | 面对裁员时会产生强烈的不公平感；倾向于采取强硬立场维护权益；可能会主动寻求法律或舆论支持 |
| **心理状态-迷茫无助** | 面对裁员时缺乏明确的应对方向；可能会依赖他人建议；需要更多的信息和支持来做出决策 |
| **沟通风格-直接强硬** | 表达意见时直截了当；面对裁员谈判时会明确提出诉求；不轻易妥协，坚持自己的立场 |
| **沟通风格-委婉协商** | 表达意见时注重方式方法；面对裁员谈判时会寻求双方都能接受的方案；愿意在合理范围内妥协 |
| **沟通风格-沉默被动** | 表达意见时较为被动；面对裁员时可能不会主动争取权益；需要外部推动才会采取行动 |
| **沟通风格-据理力争** | 表达意见时会基于事实和逻辑；面对裁员谈判时会准备充分的依据；坚持自己的合理诉求 |
| **风险承受能力-低风险** | 优先考虑尽快解决问题；对不确定性的容忍度低；倾向于接受较为保守的赔偿方案 |
| **风险承受能力-中风险** | 能够接受一定程度的不确定性；愿意通过协商寻求更好的解决方案；对维权过程有一定的心理准备 |
| **风险承受能力-高风险** | 对不确定性的容忍度高；愿意为争取合理权益投入时间和精力；可能会采取法律手段等较为激进的方式 |
