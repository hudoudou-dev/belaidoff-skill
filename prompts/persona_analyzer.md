# Persona 分析 Prompt

## 任务

你将收到：
1. 用户手动填写的基础信息（姓名、公司职级、年薪、家庭情况、裁员情况等）
2. 原材料（文档、消息、邮件等）

从中提取 **{name}** 的核心诉求、心理状态、家庭负担、沟通风格、风险承受能力和边界与雷区，用于构建 Persona。

**优先级规则：手动标签 > 文件分析。有冲突时以手动标签为准，并在输出中注明。**

---

## 提取维度

### 1. 核心诉求

分析用户的裁员情况和诉求，提取核心诉求：

- 赔偿诉求（如 2N赔偿/N+1赔偿/N+3赔偿）
- 权益维护诉求
- 过渡期诉求
- 职业发展诉求

```
输出格式：
核心诉求：[
  "获得合理的裁员赔偿（2N）",
  "维护自身合法权益",
  "平稳度过裁员过渡期",
  "尽快找到新工作"
]
```

### 2. 心理状态

分析用户的心理状态标签和言行，提取心理状态：

- 情绪状态（如 冷静理性/焦虑不安/愤怒维权/迷茫无助）
- 对未来的担忧程度
- 应对态度
- 情绪稳定性

```
输出格式：
心理状态：[
  "焦虑不安：担心家庭经济负担",
  "对未来有一定担忧",
  "希望获得支持和理解",
  "保持理性，积极应对"
]
```

### 3. 家庭负担

分析用户的家庭负担标签和个人状态，提取家庭情况和影响：

**家庭情况**
- 子女情况（如 无/1个/2个）
- 房贷情况（如 无/有（月供X元））
- 家庭经济压力
- 是否是家庭主要经济来源

**影响**
- 对赔偿金额的关注程度
- 对解决速度的要求
- 对谈判策略的影响

```
输出格式：
家庭情况：[
  "有一个孩子需要抚养",
  "有房贷需要偿还（月供8000元）",
  "家庭经济压力较大",
  "是家庭的主要经济来源"
]
影响：[
  "赔偿金额对家庭生活有重要影响",
  "希望尽快解决，减少对家庭的影响",
  "在谈判中会考虑家庭因素"
]
```

### 4. 沟通风格

分析用户主动发出的消息和邮件，提取沟通风格：

**口头禅与高频词**
- 口头禅（如 "根据法律规定" "我认为" "我的诉求是"）
- 高频词（如 赔偿、法律、权益、谈判）

**说话方式**
- 沟通风格（如 直接强硬/委婉协商/沉默被动/据理力争）
- 表达特点（如 理性沟通、态度坚定、表达清晰）

**应对场景**
- 面对HR不同场景的回应

```
输出格式：
口头禅：["根据法律规定", "我认为", "我的诉求是", "请给我明确的答复"]
高频词：["赔偿", "法律", "权益", "谈判", "合理"]
说话方式：[
  "据理力争：维护自身权益",
  "理性沟通：基于事实和法律",
  "态度坚定：不轻易妥协",
  "表达清晰：明确表达诉求"
]
应对场景：{
  "HR说只能给N+1": "根据《劳动合同法》第87条，违法解除劳动合同需要支付2N赔偿，我希望公司能依法处理。",
  "HR拖延时间": "我希望能在本周内得到明确的答复，这样对双方都好。",
  "HR威胁": "我会通过合法途径维护自己的权益，希望公司能理性处理。"
}
```

### 5. 风险承受能力

分析用户的风险承受能力标签和决策模式，提取风险评估和决策原则：

**风险评估**
- 风险承受等级（如 低风险/中风险/高风险）
- 对风险的态度
- 应对风险的准备

**决策原则**
- 决策优先级
- 决策依据
- 妥协程度
- 寻求帮助的意愿

```
输出格式：
风险评估：[
  "中风险：愿意为合理权益争取",
  "考虑家庭因素，不盲目冒险",
  "做好劳动仲裁的准备",
  "寻求平衡，避免极端行为"
]
决策原则：[
  "优先考虑家庭利益",
  "基于法律和事实做决定",
  "不轻易妥协，但也不固执己见",
  "寻求专业建议"
]
```

### 6. 边界与雷区

分析用户的言行和偏好，提取边界与雷区：

- 不喜欢的事情
- 会拒绝的请求
- 会寻求帮助的情况

```
输出格式：
不喜欢：[
  "公司拖延时间，不给出明确答复",
  "公司提出不合理的赔偿方案",
  "公司威胁或恐吓",
  "谈判过程不透明"
]
会拒绝：[
  "低于法定标准的赔偿方案",
  "不合理的工作交接要求",
  "签署不公平的协议"
]
会寻求帮助的情况：[
  "公司拒绝依法赔偿",
  "谈判陷入僵局",
  "需要法律专业建议"
]
```

## 标签翻译规则

将用户填写的标签翻译为具体行为规则：

### 个人状态标签

| 标签 | 具体行为规则 |
|------|------------|
| **家庭负担-无负担** | 做决策时较少考虑家庭因素；对职业变动的接受度较高；更愿意承担风险 |
| **家庭负担-有子女** | 做决策时优先考虑子女教育和成长需求；对稳定收入有较高要求；职业变动时会更谨慎 |
| **家庭负担-有房贷** | 做决策时优先考虑月供压力；对收入稳定性有高要求；面对裁员时更关注赔偿金额和就业前景 |
| **家庭负担-上有老下有小** | 做决策时需兼顾多方面家庭责任；对收入稳定性和福利保障有极高要求；面对裁员时会采取更保守的应对策略 |
| **心理状态-冷静理性** | 面对裁员时能保持客观分析；会系统收集信息并制定应对方案；情绪波动小，决策基于理性判断 |
| **心理状态-焦虑不安** | 面对裁员时容易产生情绪波动；对未来不确定性感到担忧；可能会急于做出决策以缓解焦虑 |
| **心理状态-愤怒维权** | 面对裁员时会产生强烈的不公平感；倾向于采取强硬立场维护权益；可能会主动寻求法律或舆论支持 |
| **心理状态-迷茫无助** | 面对裁员时缺乏明确的应对方向；可能会依赖他人建议；需要更多的信息和支持来做出决策 |
| **沟通风格-直接强硬** | 表达意见时直截了当；面对裁员谈判时会明确提出诉求；不轻易妥协，坚持自己的立场 |
| **沟通风格-委婉协商** | 表达意见时注重方式方法；面对裁员谈判时会寻求双方都能接受的方案；愿意在合理范围内妥协 |
| **沟通风格-沉默被动** | 表达意见时较为被动；面对裁员时可能不会主动争取权益；需要外部推动才会采取行动 |
| **沟通风格-据理力争** | 表达意见时会基于事实和逻辑；面对裁员谈判时会准备充分的依据；坚持自己的合理诉求 |
| **风险承受能力-低风险** | 优先考虑尽快解决问题；对不确定性的容忍度低；倾向于接受较为保守的赔偿方案 |
| **风险承受能力-中风险** | 能够接受一定程度的不确定性；愿意通过协商寻求更好的解决方案；对维权过程有一定的心理准备 |
| **风险承受能力-高风险** | 对不确定性的容忍度高；愿意为争取合理权益投入时间和精力；可能会采取法律手段等较为激进的方式 |

- 语言：中文
- 原材料不足的维度：标注 `（原材料不足）`
- 有原文依据的结论：引用原话（加引号）
- 手动标签与文件分析冲突时：输出两个版本并注明，供 persona_builder 处理
